Vue.js ilkin main.js dosyasi:
import { createApp } from "vue";

const app = createApp({
  data() {
    return {
      name: "Vue.js",
    };
  },
  template: `<h1>Bu project bir {{name}} uygulamasidir</h1>`,
});

app.mount("#app");


SFC - single file component
Single File Component (SFC) terimi, özellikle Vue.js ekosisteminde, 
bir bileşeni tek bir dosyada tanımlama ve düzenleme yöntemini ifade eder. 
Bu yöntem, bileşenlerin HTML yapısını, stilini ve JavaScript işlevselliğini 
tek bir dosyada birleştirir. SFC'ler genellikle .vue uzantısına sahiptir ve 
Vue.js uygulamalarında yaygın olarak kullanılır.


SOC - seperation of concerns
Separation of Concerns (SoC) prensibi, Vue.js projelerinde farklı işlevlerin 
ayrıştırılması ve her bir işlevin kendi sorumluluk alanında izole edilmesi anlamına gelir. 
Bu, kodun daha düzenli, okunabilir ve bakımının kolay olmasını sağlar.
